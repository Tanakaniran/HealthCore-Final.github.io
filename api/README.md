# HealthCore
Proyek ini dibangun oleh Tim Pita Hijau Pejuang, untuk kompetisi TechSprint Innovation Cup 2026 dengan Tema "Smart Digital Solution for 
Real World Problems" dan SubTema "Health & Wellbeing Technology" .

## Tentang Aplikasi
HealthCore bukan sekadar aplikasi pencatat kesehatan biasa. Kami menggabungkan pemantauan kesehatan fisik (gula darah) dan mental (detox adiksi digital) ke dalam satu ekosistem. Dimana alasan menggabungkan kedua bagian ini juga berhubungan dengan makna "Pita Hijau" yang dimana ini merupakan lambang internasional untuk melambangkan kebebasan dari penyakit fisik & mental. Dilengkapi dengan AI untuk memberikan insight real-time, HealthCore juga memiliki sistem Smart Referral Bisnis To Bisnis (B2B) yang otomatis mengarahkan pengguna ke klinik mitra jika kondisi kesehatan mereka masuk kategori berisiko atau Tinggi. HelathCore sudah melengkapi juga syarat fitur pada guidebook, salah satunya website ini sudah responsive pada perangkat lainnya

## Fitur Utama
- Diabetes AI Screening: Analisis kadar gula darah dan panduan nutrisi dan saran otomatis menggunakan AI.
- Mental Detox Journaling: Fitur pemantauan dan terapi ringan untuk membantu recovery dari Adiksi Digital/Pornografi, FOMO, dan Lingkungan Toxic
- Smart Dashboard & B2B Referral: Visualisasi data interaktif yang terhubung langsung dengan layanan rujukan klinik jika terdeteksi anomali kesehatan.

## Tech Stack
- Frontend: HTML5, Tailwind CSS (via CDN), Vanilla JavaScript, Chart.js.
- Backend & Database: Supabase (PostgreSQL) dan juga Row Level Security (RLS).
- AI Engine: Llama-3 (Large Language Model) yang terintegrasi via API(Aplication Programming Interface).

## Cara Menjalankan di Local
Aplikasi ini berjalan penuh di sisi Client-Side. Tidak perlu install node modules atau setup server yang ribet.
1. Download atau `git clone` repository ini.
2. Buka folder proyek.
3. Klik dua kali pada file `index.html` untuk membuka di browser, atau gunakan ekstensi Live Server di VS Code untuk pengalaman terbaik.

## Jika satu Folder proyek dimasukan ke vsCode
1. Masukan Folder proyek di VSCODE
2. buka terminal lalu ubah powershell menjadi github, dan klik node server. Server localhost dijalankan di server.js
3. gunakan Live Server agar cepat dan mendapatkan pengalaman terbaik.
